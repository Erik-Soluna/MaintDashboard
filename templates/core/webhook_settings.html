    Setup Instructions
    Portainer Webhook Configuration
    1. Log into your Portainer instance
    2. Go to Settings > Webhooks
    3. Click Add Webhook
    4. Configure the webhook with these settings:
    Webhook URL: {{ webhook_url }}
    HTTP Method: POST
    Content Type: application/json
    Secret: [Your configured secret]
    Events: Stack Update, Image Update
    Environment Variables

    Add these environment variables to your Django settings:

    PORTAINER_URL=http://your-portainer:9000
    PORTAINER_USER=your-username
    PORTAINER_PASSWORD=your-password
    PORTAINER_STACK_NAME=your-stack-name
    PORTAINER_WEBHOOK_SECRET=your-webhook-secret
    Testing the Webhook

    You can test the webhook manually using curl:

    curl -X POST {{ webhook_url }} \
      -H "Content-Type: application/json" \
      -H "X-Portainer-Signature: [your-signature]" \
      -d '{"event": "stack_update", "stack": "{{ stack_name }}"}'

CSV Format

Your CSV file should have the following columns:

  • name - Activity type name (required)
  • description - Description of the activity
  • estimated_duration_hours - Hours to complete (default: 1)
  • frequency_days - Days between activities (default: 30)
  • is_mandatory - true/false (default: false)
  • is_active - true/false (default: true)
